# 0x00 - 前言

​ 初学php代码审计，是自己的第一个thinkphp框架类的cms审计也是自己的第一篇博客，希望写点东西能让自己专注思路也便于做个记录后续回顾。菜鸡初上路，采取通读全文，争取读懂每一个页面的每一个操作，记录下思路和感悟养成写文章的习惯。话不多说开始吧~

# 0x01 - 安装流程

​ 入口文件index.php进入

检测/data/install目录下是否有锁文件，用此锁文件来判断是否已经安装。后续关注是否有文件删除漏洞来对锁文件进行删除，是否可以触发重装漏洞。

​ 接着重定向到安装文件，填写信息后进行安装。安装过程中有一处创建表是字符串拼接，更新系统后台管理员账号密码也是变量拼接，用的是mysqli连接，但是因为是安装步骤，一般不考虑这里有什么漏洞，无视就好。

安装过程发现几个重要的点：

1.$pass 的作用

​ $pass作用：使用硬编码加随机数作为盐，然后将明文密码连接盐进行MD5生成密文保存在数据库中。

2.写入了很多配置文件，出现了很多个以$password为前缀的配置。

​ 如Session，Cache，Cookie，都在配置文件里面写入了$password 这个5位数的随机字母。后续关注有何用处。
